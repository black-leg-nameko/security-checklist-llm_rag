# 公開リポジトリについて

## 本リポジトリの目的

本リポジトリは、LLM/RAGセキュリティ診断サービスの**公開情報**を提供するためのものです。

## 含まれる内容

以下のファイルは、サービスの透明性と信頼性を示すため、公開しております：

- **README.md**: サービス概要と診断の流れ
- **checklist.md**: セキュリティチェックリスト（診断項目の透明性確保）
- **LICENSE**: ライセンス情報
- **.gitignore**: Git除外設定

## 非公開の内容

以下の内容は、当社の知的財産として非公開となっております：

- **診断スクリプト** (`test_security.py`): 自動化された診断ツールの実装
- **使用例・サンプルコード** (`examples/`): 診断ツールの使用方法
- **依存関係ファイル** (`requirements.txt`): 診断スクリプトの依存パッケージ
- **診断レポートテンプレート**: 顧客固有のレポート形式
- **その他のプロプライエタリツール**: 当社が独自に開発した診断ツール

## 理由

診断ツールの実装詳細を公開しない理由は以下の通りです：

1. **知的財産の保護**: 診断手法やテストケースは、当社サービスの価値の一部です
2. **競合対策**: 診断手法の模倣を防ぐため
3. **差別化要因**: 独自の診断手法がサービスの差別化要因となっています
4. **顧客への価値提供**: 診断ツール自体がサービスの一部であり、有料サービスとして提供しています

## 診断サービスのご利用について

実際の診断サービスをご希望の場合は、お問い合わせください。診断ツールを使用した詳細な診断を実施し、包括的なレポートをご提供いたします。

## チェックリストの活用

公開されているチェックリストは、お客様ご自身でもご活用いただけます。ただし、専門的な診断や自動化ツールによる検証については、当社の診断サービスをご利用いただくことをお勧めいたします。
